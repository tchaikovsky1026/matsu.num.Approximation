# matsu.num.Approximation
`matsu.num.Approximation` は Java 言語向けの数値計算で使用できる, 
独自の数の型を扱うライブラリである.

現在のリリースバージョンは 24.0.0 であり, Java 17 に準拠する.
このバージョンにおいて, 次がサポートされている.

- `double` 型の入出力により表現された1変数関数の多項式近似
- 独自に定義された数体型の入出力により表現された1変数関数の多項式近似
